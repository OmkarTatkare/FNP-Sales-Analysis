# FNP-Sales-Analysis (Interactive Dashboard Creation using MS Excel)

## Project Objective
The objective of this project was to analyze FNP sales data and create an interactive Excel dashboard to monitor sales performance, customer behavior, product demand, and delivery trends. The dashboard helps businesses generate meaningful insights and improve data-driven decision-making.

## Dataset Used
- <a href="https://github.com/OmkarTatkare/FNP-Sales-Analysis/blob/main/project_fnp.xlsx">Dataset</a>

## Tools & Technologies Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning & Preprocessing

## Questions (KPIs)
- What is the total number of orders?
- What is the total revenue generated?
- What is the average customer spending?
- What is the average order-delivery time?
- Which occasions generate the highest revenue?
- Which months contribute the highest sales?
- Top 10 cities based on total orders.
- Which product categories contribute maximum revenue?
- Identify top-performing products by revenue.
- Analyze revenue trends by order time.

## Process
- Collected and verified FNP sales data.
- Performed data cleaning and preprocessing.
- Removed inconsistencies and formatted data types properly.
- Created pivot tables and pivot charts for analysis.
- Developed KPI cards including:
  - Total Orders
  - Total Revenue
  - Avg Order Delivery Time
  - Avg Customer Spend
- Designed an interactive Excel dashboard using:
  - Pivot Charts
  - Slicers
  - Dynamic Filters
  - KPI Cards
- Added filtering options for:
  - Delivery Date
  - Order Date
  - Occasion
- Built dynamic visualizations for:
  - Revenue by Occasion
  - Revenue by Month
  - Revenue by Category
  - Revenue by Hour
  - Top Products by Revenue
  - Top Cities by Orders

## Project Insights
- Anniversary and Raksha Bandhan generated the highest revenue among all occasions.
- February and August recorded peak sales performance.
- Imphal and Kavali were among the top-performing cities based on order count.
- Colors and Soft Toys contributed the highest category revenue.
- Peak order activity was observed during evening hours.
- Average customer spending was approximately ₹3,520.
- The dashboard enabled easy comparison of sales trends across months, occasions, and product categories.

## Final Conclusion
The interactive Excel dashboard provides valuable business insights into customer purchasing behavior, seasonal sales trends, and product performance. By leveraging dynamic charts, pivot tables, and slicers, businesses can optimize marketing strategies, improve product planning, and enhance customer satisfaction through data-driven decision-making.
